📚 Toko Buku API

RESTful API menggunakan Node.js dan Express.js

Pendahuluan

Project ini merupakan RESTful API untuk mengelola data buku dan transaksi pembelian buku menggunakan Node.js dan Express.js.

Judul Project & Studi Kasus

RESTful API Toko Buku Online dengan dua endpoint utama yaitu Buku dan Pembelian.

Teknologi yang Digunakan

  • Node.js
  • Express.js
  • JavaScript
  • Thunder Client
  • Visual Studio Code
  • GitHub

Cara Menjalankan Project

npm install
node server.js

TOKOBUKUAPI

Daftar Endpoint

GET /api/docs
GET /api/v1/buku
GET /api/v1/buku/:id
POST /api/v1/buku
PUT /api/v1/buku/:id
DELETE /api/v1/buku/:id
GET /api/v1/pembelian
GET /api/v1/pembelian/:id
POST /api/v1/pembelian
PUT /api/v1/pembelian/:id
DELETE /api/v1/pembelian/:id

Informasi Server

Host : localhost

Port : 3000

Framework : Express.js
